GroovyCSV by xlson

A simple CSV parsing library for groovy

GroovyCSV is a library to make csv processing just a little bit Groovier. The library uses [opencsv](http://opencsv.sourceforge.net/) behind the scenes and merely tries to add a thin layer of “Groovy-ness” to the mix.

Dependencies

* [opencsv 2.1](http://opencsv.sourceforge.net/)

Install

@Grab('com.xlson.groovycsv:groovycsv:0.2') import com.xlson.groovycsv.CsvParser def csv = '''Name-Lastname Mark-'Anderson-Nielsen' Pete-Hansen''' def data = new CsvParser().parse(csv, separator: '-', quoteChar: "'") for(line in data) { println "$line.Name $line.Lastname" }
